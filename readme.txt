== Description ==

This is a simple plugin for adding custom nav menus to your WordPress Adminbar/Toolbar. Please note this: **WP Adminbar Menus** plugin is only compatible for **WordPress 4.6** or above versions. Please update your WordPress application if you're using an old version.

**Features**  

* WordPress Multisite Compatible.
* More WordPress admin menus shortcut links under "Site Menu" & "Add New" section in Adminbar. 

== Installation ==

1. Upload **WP Adminbar Menus** folder to the ‘/wp-content/plugins/‘ directory or download it from the administration
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate in Wordpress Admin Pages and see the new, clean design!
